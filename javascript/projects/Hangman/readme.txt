## Project Description
Hangman is a word guessing game where the player must guess a word by suggesting letters one at a time. If the player guesses a letter that is not in the word, they get closer to losing the game. The goal is to guess all the letters in the word before running out of attempts.

## Key Features
- Ability to choose a category for the words.
- Score tracking system.
- User-friendly interface.
- Graphical display of the hangman, progressing with incorrect guesses.
- Supports multiple words and replayability.
