Snake Game
Overview
This project is a classic Snake game developed as part of a JavaScript module assignment. The game is designed to help users practice their coding skills and understand the logic behind game development using JavaScript.

Features
Classic Snake gameplay.
Responsive design to fit various screen sizes.
Score tracking.
Simple, clean, and intuitive UI.
Technologies Used
HTML5: For the structure of the game.
CSS3: For styling the game interface.
JavaScript (ES6): The core logic and interactivity of the game.

How to Play
Use the arrow keys on your keyboard to control the direction of the snake.
The goal is to eat as many food items as possible without running into the walls or the snake's own tail.
Each time the snake eats a piece of food, it grows longer, and the game becomes more challenging.
