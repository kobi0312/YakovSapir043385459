# Calculator Project

This is a simple calculator project built as part of a JavaScript module. The calculator can perform basic arithmetic operations such as addition, subtraction, multiplication, and division.

## Overview
This project is a fully functional calculator developed using HTML, CSS, and JavaScript. The calculator has a user-friendly interface and can handle basic mathematical operations. This project is aimed at improving my skills in front-end development, particularly in JavaScript.

## Features
- Basic arithmetic operations: addition, subtraction, multiplication, and division.
- Clear button to reset the calculator.
- Responsive design that works well on different screen sizes.
- Error handling for division by zero.

## Technologies Used
- **HTML5**: For structuring the web page.
- **CSS3**: For styling and layout.
- **JavaScript (ES6+)**: For calculator logic and interactions.
