Puzzle Game - JavaScript Project
Overview
Welcome to the Puzzle Game! This project is a part of my studies in the JavaScript module. The game is designed to challenge players to solve a puzzle by rearranging pieces into the correct order.

Features
Drag and drop functionality to move puzzle pieces.
Different levels of difficulty.
Timer to track how long it takes to solve the puzzle.
Responsive design that works on various screen sizes.
Technologies Used
HTML5: For structuring the game layout.
CSS3: For styling the game interface and making it responsive.
JavaScript: For the game logic, including handling user interactions, checking the puzzle's completion, and updating the game state.

How to Play
When the game starts, you'll see a scrambled puzzle.
Use drag and drop to move the pieces into the correct position.
Once all pieces are in the correct order, the puzzle will be solved, and you'll receive a success message!
