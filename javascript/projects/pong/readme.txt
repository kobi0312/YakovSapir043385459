Pong Game
Author: Kobi Sapir

Project Description
This project is a classic Pong game built as part of my studies in a JavaScript module. The game is developed using HTML, CSS, and vanilla JavaScript. The objective of the game is to hit the ball back and forth between two paddles and try to score points against the opponent.

Features
Single Player Mode: Play against the computer with increasing difficulty levels.
Responsive Design: The game adapts to different screen sizes, ensuring a smooth experience on both desktop and mobile devices.
Customizable Settings: Players can adjust the speed of the ball and the size of the paddles.
Score Tracking: The game keeps track of the score for both players.
