# Tetris Game

## Project Overview

Welcome to my Tetris game project! This project was created as part of my JavaScript module for my studies. The game is a classic implementation of Tetris, built using HTML, CSS, and JavaScript.

## Features

- **Classic Tetris Gameplay:** Enjoy the traditional Tetris experience with all the standard features, including rotating pieces, clearing lines, and increasing difficulty.
- **Responsive Design:** The game is fully responsive and can be played on various screen sizes, from desktops to mobile devices.
- **Score Tracking:** Keep track of your score as you progress through the levels.
- **Pause/Resume Functionality:** You can pause and resume the game at any time.
- **Sound Effects:** Includes sound effects for piece movements, line clears, and more.

## Technologies Used

- **HTML5:** For the structure of the game.
- **CSS3:** For styling and layout.
- **JavaScript (ES6+):** For game logic and interactivity.
